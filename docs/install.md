PIjs是一个可执行文件，可以采用以下几种方式安装

## 安装

### Linux

curl 安装

```
curl -fsSL https://github/pijs/install | bash
```

### windows and linux
npm 安装

```
npm install -g pijs
```

### Dokcer

```
docker pull xxx/pijs
docker run --rm --init --ulimit memlock=-1:-1 xxx/pijs
```

### 检测安装版本

```
pijs --version
0.1.x
```
